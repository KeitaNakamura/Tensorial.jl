Levi-Civita

Tensorial.levicivita — Function
levicivita(::Val{N} = Val(3))

Return N dimensional Levi-Civita tensor.

Examples

julia> ϵ = levicivita()

[:, :, 1] =

 0   0  1

[:, :, 2] =

 0  0   0
 1  0   0

[:, :, 3] =

 -1  0  0
  0  0  0
